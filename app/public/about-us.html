Introduction of Website

Subedi Thapa Wardrobe Rental is a student cloud web application created by Ashish Subedi and Bijay Thapa. The website presents a simple online clothing rental idea where customers can browse traditional and modern outfits, check rental prices and access contact details. The project is designed to show static website hosting, Node.js Express, Terraform Infrastructure as Code and GitHub Actions deployment on Microsoft Azure.
